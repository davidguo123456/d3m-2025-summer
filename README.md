# Start Guide

Install [nvm](https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating)

```
nvm install v24
nvm use v24

:: use this to set your default ver
nvm alias default v24 

node -v
```

In project folder

```
npm install
npm run dev
```
